# Jayasom Wireframes — V3

Working set of low-fidelity wireframes for the Jayasom AMAALA website. Each page is a single self-contained HTML file using the locked Jayasom design system (no build step, no images, no external dependencies).

## Getting started

Open `index.html` in any browser to see the page list, or open any wireframe file directly.

```
.
├── index.html                          # Page index (links public pages only)
├── wireframe-homepage-v3.html
├── wireframe-about-us-v3.html
├── wireframe-as-a-woman-v3.html
├── wireframe-as-a-pair-v3.html
├── wireframe-as-a-family-v3.html
├── wireframe-as-a-leader-v3.html       # Not linked from index
└── wireframe-as-a-flow-v3.html         # Not linked from index
```

## Pages

### Listed on the index

- **Homepage** — brand entry point
- **About Us** — origin, philosophy, founders, pillars
- **As a Woman** — retreat journey for women
- **As a Pair** — shared retreat journey for couples
- **As a Family** — multigenerational stays

### Not listed

`wireframe-as-a-leader-v3.html` and `wireframe-as-a-flow-v3.html` are part of the project but intentionally not linked from the index. They will be added later.

## Design system

All pages share the same locked CSS tokens (background, foreground, muted, border, wire-placeholder, etc.) defined inline at the top of each file. Do not modify the token values — keep them identical across pages so the set stays visually consistent.
